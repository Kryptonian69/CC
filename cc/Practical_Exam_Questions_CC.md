## 1. Basic Web Services (Logic & Arithmetic)
*   **Currency Conversion:** Implement a web service to convert Rupees to Dollar (1 USD = 75 INR). 
    *   **Call from:** .NET / JAVA
    *   **Source:** pdf1 (Page 1)
*   **Temperature Conversion:** Implement a web service to convert Fahrenheit to Celsius ($C = (F - 32) \times 5/9$).
    *   **Call from:** JAVA / .NET
    *   **Source:** pdf1 (Page 2)
*   **Factorial Calculation:** Implement a web service to calculate the factorial of a given number.
    *   **Call from:** JAVA / .NET
    *   **Source:** pdf1 (Page 3)
*   **Prime Number Checker:** Implement a web service to identify whether a number is prime or not.
    *   **Call from:** JAVA / .NET
    *   **Source:** pdf1 (Page 4)
